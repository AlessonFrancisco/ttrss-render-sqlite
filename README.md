# TT-RSS com SQLite para Render

Este repositório permite rodar o Tiny Tiny RSS (TT-RSS) com SQLite no Render.com.

## Etapas para deploy:

1. Faça fork deste repositório no seu GitHub
2. No Render, clique em "New Web Service"
3. Conecte sua conta GitHub
4. Selecione este repositório
5. Configure:
   - Environment: Docker
   - Branch: `main`
   - Name: como preferir
   - Start Command: (deixe em branco)

Depois de publicado, acesse a URL pública para finalizar a instalação do TT-RSS.
